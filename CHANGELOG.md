# 更新日志

本文件记录 SayIt 各版本的功能变更、Bug 修复与优化。

格式参考 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/)，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

变更分类：`新增` / `修复` / `优化` / `变更` / `移除` / `安全`。

---

## [0.0.9] - 2026-07-15

### 优化
- 优化云 API 语音识别配置：App ID 仅在豆包 ASR 下显示，千问 ASR（含流式模式）只需填写百炼 API Key。
- 优化千问 ASR 流式连接与结果聚合，避免实时连接异常后回退到录完再识别。

---

## [0.0.8] - 2026-07-09

### 优化
- 切换到云 API / 服务器模式后自动释放本地识别模型占用的内存，不再需要退出应用才能释放。
- 加快多处设置按钮的响应速度：语音识别与 AI 供应商的保存/测试连接、本地模型切换等操作，减少不必要的等待与重复请求。
- 大幅提升标题栏「AI 整理」开关与润色模式切换的响应速度，连续快速切换不再卡顿。
- 下载更新弹窗改为展示真实下载进度（按已下载字节数计算百分比），不再是固定动画。

### 新增
- 新增「插入文本后保护剪贴板」开关（默认开启）：插入完成后自动还原为插入前的剪贴板内容，避免占用剪贴板。

### 修复
- 提升服务器模式连接稳定性：修复长时间挂后台或空闲时，连接偶发反复中断、需要频繁自动重连的问题。
- 修复更新安装完成后不会自动重新打开应用的问题：现在会在安装完成后自动拉起新版本并跳转到「关于」页确认更新已生效。
- 修复应用启动自动检查更新与手动打开「关于」页检查更新可能同时触发、导致重复下载/安装的问题。
- 修复开启「录音时静音系统声音」后，就绪提示音偶发无声（首次或长时间未使用后）的问题。
- 修复远程桌面场景下误触发单键免提（Alt/Shift）的问题。

## [0.0.8] - 2026-07-08

### 新增
- 新增「录音时静音系统声音」开关（默认关闭）：按住说话时临时静音系统输出，避免外放声音被麦克风录入，松开即恢复。
- 润色模式支持自定义快捷键切换：为任意润色模式设置组合键（如 Alt+1），全局按下即可切换，切换时悬浮窗即时提示。
- 文本替换支持批量添加：粘贴多行内容（支持逗号、制表符、`=>` 分隔）一次性导入多条规则，已存在的原文会更新。
- 热键支持更多按键：菜单键、Pause、ScrollLock、Insert，便于避开易冲突的按键。
- 免提组合键（如 Alt+字母）设置时增加冲突提示，若已被其他程序占用会提醒更换。
- 关于页面新增「本次更新」板块，展示当前版本的主要更新内容。
- 服务器模式的服务地址增加说明提示（悬停信息图标查看），介绍默认体验服务器的用途与替代方案。
- 本地识别新增 Qwen3-ASR 0.6B 模型：支持多语种、准确度更高，适合对识别质量要求高的场景。
- 本地模型列表新增速度 / 准确度评分与「推荐」标识，方便按需选择合适的模型。
- 本地识别支持热词：使用 Qwen3-ASR 时，热词可提升专有名词与行业术语的识别准确率。
- 云 API 语音识别新增「小米 MiMo」供应商（mimo-v2.5-asr）：中英文与多种方言，自动标点，仅需填入 API Key。
- AI 润色新增「小米 MiMo」供应商（mimo-v2.5 / mimo-v2.5-pro），默认使用 mimo-v2.5。

### 优化
- 去除悬浮窗胶囊与卡片的投影，消除窗口边缘出现的灰色残影。
- 标题栏「AI 整理」按钮 hover 时显示当前润色模式；快捷键切换后 AI 整理页高亮实时更新。
- 意见反馈附带更多排错信息（当前应用/窗口/进程、识别参数、客户端环境），随「最后的转录」开关控制是否附带；不含音频与 IP。
- 首次安装直接进入设置向导，不再短暂闪现主页。
- 服务器模式下未连接后端时，按免提键改为显示简短告警，不再弹出无法关闭的悬浮窗。
- 托盘图标提示语更新为「随口说，出色写」。
- 长语音（超过约 30 秒）本地识别自动在停顿处分段处理，避免超长音频卡死或识别异常。
- 本地模型下载更稳更快：支持多下载源自动切换、断点续传，国内网络自动走加速通道。
- 小米 MiMo AI 润色默认关闭深度思考，回复更快。
- AI 供应商点击模型标签即刻生效并保存，无需再点保存按钮。

### 修复
- 修复在 CMD 中运行 Claude Code 等程序时无法插入文本的问题。
- 修复开启「录音时静音系统声音」后，按下热键没有就绪提示音的问题。
- 修复设置向导中更换热键后，测试仍需按默认键才生效的问题。
- 修复设置向导中更换免提热键后，首页提示仍显示旧按键、需切换页面才刷新的问题。
- 修复云 API 语音识别切换供应商时误带入其它供应商 API Key 的问题，各供应商密钥现独立保存。
- 修复 AI 润色实际运行、但未改动文本时，历史记录不显示所用 AI 供应商与模型的问题。
- 修复修改服务地址（含填错地址）后，左下角连接状态未刷新、仍显示已连接的问题。
- 修复同时打开两个客户端导致插入两份重复文本的问题，现改为单实例，二次启动会唤起已有窗口。
- 修复远程桌面等场景下，历史记录音频播放进度条卡住不动的问题。
- 修复切换 AI 供应商后可能仍沿用上一个供应商地址与模型的问题（如选了豆包却调用了其他模型）。
- 修复在 TRAE、Cursor、Windsurf 等编辑器中无法插入文本、只弹出兜底卡片的问题。
- 默认开启麦克风降噪，消除部分机器录音中的低频电流声/底噪。
